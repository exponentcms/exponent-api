Class expPaginator

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

This is the class expPaginator Exponent Pagination Subsystem

The expPaginator class is used to retrieve objects from the database and paginate them. It automagically handles the calls to other pages and had built-in sorting using the defined column headers.
