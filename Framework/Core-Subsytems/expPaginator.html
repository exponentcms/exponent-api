Class expPaginator

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

Exponent Pagination Subsystem

The expPaginator class is used to retrieve objects from the database and paginate them. It automagically handles the calls to other pages and had built-in sorting using the defined column headers.

Usage Example:

  1.  $page = new expPaginator(array(
  2.       'model'=>'faq',
  3.       'where'=>1,
  4.       'limit'=>25,
  5.       'order'=>'rank',
  6.       'controller'=>$this->baseclassname,
  7.       'action'=>$this->params['action'],
  8.       'columns'=>array('In FAQ'=>'include_in_faq', 'Submitted'=>'created_at', 'Submitted By'=>'submitter_name'),
  9.   ));
