    This converts the sql statement into a nice CSV.

    sql2csv(mixed $items, null $rptcols) : string
    Static
    We grab the items array which is stored funkily in the DB in an associative array when we pull it. So basically our aray looks like this: ITEMS {[id]=>myID, [Name]=>name, [Address]=>myaddr} {[id]=>myID1, [Name]=>name1, [Address]=>myaddr1} {[id]=>myID2, [Name]=>name2, [Address]=>myaddr2} {[id]=>myID3, [Name]=>name3, [Address]=>myaddr3} {[id]=>myID4, [Name]=>name4, [Address]=>myaddr4} {[id]=>myID5, [Name]=>name5, [Address]=>myaddr5} So by nature of the array, the keys are repetated in each line (id, name, etc) So if we want to make a header row, we just run through once at the beginning and use the array_keys function to strip out a functional header
