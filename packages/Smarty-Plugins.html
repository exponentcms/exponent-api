Smarty_Compiler_Switch¶

Switch statement plugin for smarty.

This smarty plugin provides php switch statement functionality in smarty tags. To install this plugin drop it into your smarty plugins folder. You will also need to manually load the plugin sot hat all the hooks are registered properly. Add the following line after you load smarty and create an instance of it in your source code. <code> $this->smartyObj->loadPlugin('smarty_compiler_switch'); </code>
